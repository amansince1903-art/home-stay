# Images Folder

## How to Add the Hero Background Image

1. Save your haveli/heritage hotel image as: `hero-bg.jpg`
2. Place it in this folder: `client/src/assets/images/hero-bg.jpg`
3. The CSS is already updated to use this local image

## Current Images:
- `hero-bg.jpg` - Main hero section background (haveli courtyard image)

## Image Guidelines:
- Format: JPG or PNG
- Recommended size: 1920x1080 or larger
- Optimize for web (compress to reduce file size)
- Keep file size under 500KB for better performance
